в рабочем столе в папке config - создаем файл config.json
внутри :

{
    "database": {
      "name": "server",
      "user": "postgres",
      "password": "musakhan003"
    },
    "other_settings": {
      "SOME_SETTING": "value"
    }
  тут будут настройки бд. сохроняем

pip install -r requiremets.txt
python manage.py makemigartions
python manage.py migarte
